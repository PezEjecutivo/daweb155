Como inicializar un modulo en node

Una API REST la podéis hacer con:
    - Python
    - Java con Spring Boot
    - Nodejs + Express

Como se inicializaba proyecto con npm:
    - npm init == Inicia el proyecto y nosotros configuramos/aceptamos las cosas
    - npm init -y == Inicia el proycto y le da si a todo

Como se añade una libreria:
    - npm install <libreria> == Añade la libreria
